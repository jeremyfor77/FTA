Zombie Diggers Banned by Executive Order, FTA Bunker Opens for Business

The clan's first dedicated ground: a 7 Days to Die V3.0 "Dead Hot Summer" server, hand-tuned, modded, and — per new policy — completely undiggable from below.

Field Report — FTA7D2D30NOZombieDiggers — Filed 3 July 2026

The Dispatch

After a lengthy provisioning campaign, the clan's 7 Days to Die server is standing and fully modded on Indifferent Broccoli infrastructure. The build tracks V3.0.0 "Dead Hot Summer," the game's largest sandbox overhaul to date, replacing the old scattering of individual difficulty knobs with a single portable SandboxCode string — ours is tuned for one very specific grievance: the undead can no longer dig.

Anti-cheat runs disabled fleet-wide, a requirement for the mod loadout below, all of which patch live game code rather than sitting quietly in XML. Every mod was verified line-by-line against the server's own boot log before being cleared for the roster — no silent failures, no half-loaded features.

Connection details and the current mod package live in the Comms column. New arrivals should expect an AntiCheat Incompatible Mods prompt on first join — click Restart, not Continue, or the mods simply won't load client-side.
